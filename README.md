# 批量重命名工具

一个用于批量清理 PDF 文件冗余前缀的 Python 小工具，专为整理 A 股上市公司年度报告等从数据平台下载的文件而设计。

## 背景

从 Wind、东方财富等平台下载的研究报告，文件名通常带有冗长的前缀，例如：

```
2024-04-30-600519.SH-贵州茅台-600519贵州茅台股份有限公司2024年年度报告.pdf
```

本工具可以自动识别并剥离这些冗余前缀，统一替换为你自定义的简洁格式，例如：

```
600519-贵州茅台-2024年年度报告.pdf
```

## 使用方法

**第一步：修改目标目录**

打开脚本，将第 5 行的 `folder_path` 改为你自己的文件夹路径：

```python
folder_path = r"D:\你的目标文件夹路径"
```

**第二步：运行脚本**

```bash
python change_filesname.py
```

**第三步：按提示操作**

1. 输入重命名后的文件名前缀（可留空表示不加前缀）
   - 示例：`600519-贵州茅台-` 或 `贵州茅台_` 或直接回车留空
2. 程序会列出所有文件的**重命名预览**，确认无误后输入 `y` 执行

## 注意事项

- 本工具仅处理 `.pdf` 文件，其他格式文件不受影响
- 若目标文件名已存在，该文件会自动跳过，不会覆盖
- 建议执行前先备份文件夹，以防误操作

## 环境要求

- Python 3.x
- 无需安装任何第三方库，使用标准库 `os` 和 `re` 即可

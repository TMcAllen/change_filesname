# 批量文件重命名工具 v1.1

一个带图形界面（GUI）的 PDF 批量重命名工具，能自动识别文件名中的公共前缀、公共后缀与可变部分，让你直接在界面上修改命名格式，实时预览效果后一键执行。

## 功能亮点

- **自动分析命名规律**：加载文件夹后，程序自动对比所有 PDF 文件名，识别出公共前缀、公共后缀与每个文件的可变部分（如年份、期数等）
- **可视化预览**：修改前缀/后缀时，右侧列表实时刷新"改前 → 改后"对照，所见即所得
- **智能边界识别**：自动回退到语义边界，避免把年份、编号等可变内容误归入公共前缀
- **安全执行**：文件名冲突时自动跳过，执行前弹窗二次确认，不会误覆盖已有文件
- **图形界面**：基于 tkinter，无需命令行，双击运行即可使用

## 使用方法

**第一步：运行程序**

```bash
python change_filesname_v1.1.py
```

**第二步：选择文件夹**

点击"选择文件夹"按钮，选择存放 PDF 文件的目录。程序会自动分析所有文件名，并在"识别到的命名规则"栏显示分析结果，格式如下：

```
{公共前缀}  +  【可变部分示例1/示例2/...】  +  {公共后缀}
```

**第三步：修改命名格式**

在"修改公共前缀为"和"修改公共后缀为"两个输入框中，填入你想要的新前缀/后缀。两个框均可留空。下方预览列表会**实时更新**，显示每个文件改前和改后的对照。

**第四步：确认并执行**

确认预览无误后，点击右下角"批量修改"按钮，弹窗二次确认后执行。执行完毕会弹出结果摘要。

## 使用示例

原始文件名（从数据平台下载）：

```
2024-04-30-600519.SH-贵州茅台-600519贵州茅台股份有限公司2022年年度报告.pdf
2024-04-30-600519.SH-贵州茅台-600519贵州茅台股份有限公司2023年年度报告.pdf
2024-04-30-600519.SH-贵州茅台-600519贵州茅台股份有限公司2024年年度报告.pdf
```

程序自动识别：公共前缀 `贵州茅台股份有限公司`，可变部分 `2022年/2023年/2024年`，公共后缀 `年度报告`

将前缀改为 `600519-贵州茅台-`，后缀保持不变，结果：

```
600519-贵州茅台-2022年年度报告.pdf
600519-贵州茅台-2023年年度报告.pdf
600519-贵州茅台-2024年年度报告.pdf
```

## 注意事项

- 本工具仅处理 `.pdf` 文件，其他格式文件不受影响
- **可变部分**（如年份、期数）由程序自动识别并保留，重命名时不会被修改
- 若目标文件名已存在，该文件自动跳过，不会覆盖
- 建议执行前先备份文件夹，以防误操作
- 重命名操作执行后**无法通过程序撤销**，请仔细核对预览后再点击执行

## 环境要求

- Python 3.x
- 无需安装任何第三方库，使用标准库 `os`、`re`、`tkinter` 即可
